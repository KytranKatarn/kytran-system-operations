Add New Physical Volume (expand VG)

To add more space to your volume group, you need a new disk or partition. Steps:

  1. Add a new disk or create a new partition
  2. Initialize as PV: sudo pvcreate /dev/sdX
  3. Add to VG: sudo vgextend ubuntu-vg /dev/sdX
  4. Then return here to extend the LV
If your disk has unallocated space, you can grow the existing partition with growpart or fdisk, then run pvresize /dev/sdaX
